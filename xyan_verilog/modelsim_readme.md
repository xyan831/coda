# setup modelsim for verilog simulation

download ModelSimSetup
```
https://www.altera.com/downloads/simulation-tools/modelsim-fpgas-standard-edition-software-version-20-1-1
```

setup ModelSim - choose Starter Edition (no license)

# simulation files

Launch_ModelSim: run Modelsim (edit as txt to change modelsim.exe path)

runlab.do: edit to simulate and check waveform

# how to use ModelSim

run command to start simulation
```
do runlab.do
```

add waveform: choose signals in Objects window >> Ctrl+W

save waveform format: click Wave window >> File >> Save Format...
